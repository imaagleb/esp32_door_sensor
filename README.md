# esp32_door_sensor
__ENG__\
ESP32-Based Automatic Door Sensor (18650 Powered)\
__RUS__
Это мой pet-project для ESP32, который будет работать с HomeKit от Apple для отображения состояния двери (открыто/закрыто) с использованием геркона. Также я реализовал deep sleep, чтобы мк просыпался только при изменении состояния на пине D12 (к которому подключен геркон). Это делает датчик безумно автономным, потребление тока при режиме deep sleep составляет 10 µA - 150 µA.

Для работы с HomeKit мы будем использовать библиотеку esp-homekit-sdk. Убедитесь, что она добавлена в вашу среду разработки.
